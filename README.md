# GateMotorController
A controller to be used with a gate motor

#Material:
 - 1 Photocell gate kit
 - 1 Power supply with output of 12V
 - 5 conectors with 2 pins each to solder in PCB
 - Prototype board 9X15 cm
 - 433 Mhz reciever
 - 433 Mhz remote
 - 2A 5V ac adapter
 - module of 4 relays
 - WeMos d1 mini
 - 2 gate motors
